# Эдгар

2. Эдгар имеет среднее количество здоровья, очень высокую скорость передвижения, очень высокую скорость перезарядки, малую дальность атаки (тк является боижним бойцом) и довольно большой урон. Особенность позволяет Эдгару заряжать свой Супер автоматически.
   # *Характеристики*
   - эпический боец
   - супер: прыжок через препятствия с нанесением урона при приземлении
   - пассивное восстановление здоровья при попадании по врагам
   - гиперзаряд: "взрывная реакция"
